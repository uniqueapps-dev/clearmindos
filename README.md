# clearmindos

ClearMindOS (CmOS)

ClearMindOS is a behavioral operating system designed to help you think clearly, act decisively, and build discipline through structured daily reflection.

It is not a note-taking app, not a quote generator, and not a productivity tracker.

It is a daily feedback system.

---

🧠 Core Idea

ClearMindOS runs on a simple but powerful loop:

1. Direction (Morning) — Define what matters
2. Correction (Midday / Struggle) — Break resistance
3. Accountability (Evening) — Review execution
4. Reflection (Meditate) — Reset and think clearly

Over time, it tracks patterns, highlights inconsistencies, and pushes toward action.

---

⚙️ System Architecture

- Single API (OpenRouter)
- Two AI Engines:
  - Stoic Engine → Insight grounded in Meditations by Marcus Aurelius
  - Behavior Engine → Pattern detection and corrective feedback

---

🔁 Key Features

- Daily structured flow (Morning → Midday → Evening)
- AI-powered behavioral feedback
- Context-aware one-line signals
- Theme progression (control, discipline, ego, time, mortality, duty)
- Local memory (last 7 sessions)
- Evening accountability system
- Personal greeting and adaptive tone
- Lightweight, no backend required

---

🎯 Purpose

Most apps give information.

ClearMindOS is built to change behavior.
